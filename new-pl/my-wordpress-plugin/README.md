# My WordPress Plugin

## Description
This plugin adds a simple admin page to the WordPress dashboard that displays a welcome message. It is designed to be lightweight and easy to use.

## Installation
1. Upload the `my-wordpress-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

## Usage
Once activated, you will see a new menu item in the WordPress admin sidebar. Click on it to view the welcome message.

## Features
- Adds a custom admin page.
- Displays a welcome message.
- Simple and easy to use.
